Meshtastic Menu Tree

Meshtastic App Top-Level (Android)
  • Connection
    • Bluetooth Pairing/Selection
      Pair and select Bluetooth devices for connection to Meshtastic hardware. Default: N/A
    • WiFi/IP Connection
      Connect to Meshtastic devices via WiFi or Ethernet IP address. Default: N/A
    • Serial/USB Connection
      Connect via serial USB for direct hardware access, useful for debugging or CLI. Default: N/A
  • Messages
    • Threaded Conversations
      View and participate in threaded message conversations across the mesh. Default: N/A
    • Quick Chat Actions (e.g., thumbs up, location share)
      Quick actions like reactions (thumbs up) or sharing location in chats. Default: N/A
  • Map
    • Node Locations
      Display locations of nodes on the map based on shared positions. Default: N/A
    • Waypoints
      Create and view waypoints for navigation or points of interest. Default: N/A
    • Offline Maps Download
      Download maps for offline use in areas without internet. Default: N/A
  • Nodes
    • Node List (online/offline, SNR, hops)
      List of nodes with status (online/offline), signal-to-noise ratio (SNR), and hop count. Default: N/A
    • Remote Admin (via Admin Channel)
      Administer remote nodes via the Admin channel for configuration changes. Default: N/A
  • Settings
    • App Settings
      • Theme (Light, Dark, System)
        Choose app theme: Light, Dark, or follow System settings. Default: System
      • Map Settings (Online/Offline, Tile Sources)
        Configure map type (online/offline) and tile sources for custom maps. Default: Online
      • Message Notification Settings
        Customize notifications for incoming messages. Default: Enable
      • Debug Logs
        View and export debug logs for troubleshooting. Default: N/A
      • Device DB Cache Limit
        Set limit for device database cache to manage storage. Default: Unlimited
      • System Settings (Permissions)
        Manage system permissions like location, Bluetooth, etc. Default: N/A
    • Radio Configuration
      • User
        • Long Name
          A personalized name for your device. Auto-generated by default. If you are a licensed HAM operator and have enabled IsLicensed, this should be set to your HAM operator call sign. Default: Generated
        • Short Name
          A personalized short identifier for your device. Auto-generated by default. Default: 4 chars
        • Team
          Team color for ATAK integration, to categorize nodes in teams. Default: None
        • Is Licensed Operator
          If you are a licensed HAM operator and have considered the privileges and restrictions of using Meshtastic with a HAM license, enable this flag. Disabled by default. Default: Disable
        • MAC Address
          MAC Address of the device, read-only. Default: N/A
      • Channels (Up to 8 channels)
        • Channel List
          • For each channel:
            • Name
              A short identifier for the channel. (< 12 bytes) Default: Empty
            • Role
              Role of the channel (enum: DISABLED, PRIMARY, SECONDARY). Default: PRIMARY for channel 0
            • PSK
              The encryption key used for private channels. Hex byte 0x01 for the Primary default channel. Must be either 0 bytes (no crypto), 16 bytes (AES128), or 32 bytes (AES256). Default: Random
            • Uplink Enabled
              Bool: Send mesh messages to internet via MQTT gateway (default: false). Default: Disable
            • Downlink Enabled
              Bool: Receive internet messages via MQTT gateway to local mesh (default: false). Default: Disable
            • Modem Preset
              Enum: Inherit from LoRa preset or custom. Default: Inherit
            • Module Permissions
              Bitfield: Permissions for modules like Telemetry on this channel. Default: All allowed
        • QR Code/Share URL for Channel
          Generate QR code or URL to share channel settings easily. Default: N/A
      • Device
        • Role
          Enum: CLIENT (standard messaging), ROUTER (rebroadcasts), REPEATER (simple relay), TRACKER (GPS focus), etc. Defines core behavior. Default: CLIENT
        • Rebroadcast Mode
          Enum: ALL (all messages), LOCAL_ONLY (only local), KNOWN_ONLY (favorited nodes), etc. Controls rebroadcast. Default: ALL
        • Button GPIO
          uint32: GPIO pin for user button if not predefined. Default: Hardware-specific
        • Buzzer GPIO
          uint32: GPIO pin for PWM buzzer if not predefined. Default: Hardware-specific
        • Buzzer Mode
          Enum: ALL_ENABLED (full use), DISABLED, NOTIFICATIONS_ONLY, etc. Controls buzzer behavior. Default: ALL_ENABLED
        • Node Info Broadcast Secs
          uint32: Seconds between NodeInfo broadcasts (includes name info). Default: 900
        • Double Tap as Button Press
          Bool: Treat double tap on accelerometer as button press. Default: Disable
        • Disable Triple Click
          Bool: Disable triple click for GPS functions. Default: Disable
        • TZDEF
          String: POSIX timezone format for device time (e.g., 'EST5EDT'). Default: UTC
        • LED Heartbeat Disabled
          Bool: Disable LED heartbeat indicator. Default: Disable
      • Position
        • Position Broadcast Secs
          uint32: Broadcast position interval in seconds if not smart. Default: 900
        • Smart Position Enabled
          Bool: Enable smart position sending based on movement (reduces unnecessary broadcasts). Default: Enable
        • Fixed Position
          Bool: Set node as fixed position (no movement). Default: Disable
        • GPS Mode
          Enum: DISABLED, ENABLED, NOT_PRESENT. Controls GPS hardware use. Default: ENABLED if present
        • GPS Update Interval
          uint32: GPS position attempt interval. Default: 30
        • Position Flags
          Bitfield: Include ALTITUDE, DOP, HEADING, etc., in position packets. Default: ALTITUDE | DOP | HEADING | SPEED
        • RX GPIO
          uint32: GPIO for GPS RX. Default: Hardware-specific
        • TX GPIO
          uint32: GPIO for GPS TX. Default: Hardware-specific
        • Broadcast Smart Min Distance
          uint32: Min distance (meters) for smart broadcast trigger. Default: 30
        • Broadcast Smart Min Interval Secs
          uint32: Min interval (seconds) for smart broadcast trigger. Default: 120
        • GPS EN GPIO
          uint32: GPIO to enable GPS power. Default: Hardware-specific
      • Power
        • Is Power Saving
          Bool: Enable power saving by disabling BT, WiFi, screen, etc. Default: Disable
        • On Battery Shutdown After Secs
          uint32: Shutdown after this time on battery loss. Default: 0
        • ADC Multiplier Override
          Float: Override voltage divider ratio (e.g., 3.20). Default: Hardware-specific
        • Wait Bluetooth Secs
          uint32: Bluetooth timeout before sleep. Default: 60
        • SDS Secs
          uint32: Super deep sleep interval for extreme power saving. Default: 0
        • LS Secs
          uint32: Light sleep interval for moderate saving. Default: 300
        • Min Wake Secs
          uint32: Minimum wake time before sleep. Default: 10
        • Device Battery INA Address
          uint32: I2C address for INA battery monitor. Default: 0x40 if present
        • Powermon Enables
          uint64 bitfield: Enable specific power monitoring features. Default: 0
      • Network
        • WiFi Enabled
          Bool: Enable WiFi for IP-based connections. Default: Disable
        • WiFi SSID
          String: WiFi network SSID. Default: Empty
        • WiFi PSK
          String: WiFi network password. Default: Empty
        • NTP Server
          String: NTP server for time sync. Default: meshtastic.pool.ntp.org
        • ETH Enabled
          Bool: Enable Ethernet if hardware supports. Default: Disable
        • Address Mode
          Enum: DHCP (auto IP), STATIC (manual). Default: DHCP
        • IPv4 Config
          Sub-settings: IP, Gateway, Subnet, DNS for static IPv4. Default: DHCP values
        • Rsyslog Server
          String: Server for remote syslogs. Default: Empty
        • Enabled Protocols
          uint32 bitfield: Enable protocols like UDP_BROADCAST. Default: 0
        • IPv6 Enabled
          Bool: Enable IPv6 support. Default: Disable
      • Display
        • Screen On Secs
          uint32: Screen timeout after activity. Default: 60
        • Auto Screen Carousel Secs
          uint32: Auto-rotate screen pages interval. Default: 5
        • Flip Screen
          Bool: Invert screen orientation. Default: Disable
        • Units
          Enum: METRIC or IMPERIAL units. Default: METRIC
        • OLED Type
          Enum: AUTO-detect, SSD1306, SH1106, etc., for OLED type. Default: AUTO
        • Display Mode
          Enum: DEFAULT, TWOCOLOR, INVERTED, COLOR for display mode. Default: DEFAULT
        • Heading Bold
          Bool: Bold heading text. Default: Enable
        • Wake On Tap Or Motion
          Bool: Wake screen on tap or motion. Default: Enable
        • Compass Orientation
          Enum: DEGREES_0 etc., for compass direction. Default: DEGREES_0
        • Use 12h Clock
          Bool: Use 12-hour clock format. Default: Disable
        • Use Long Node Name
          Bool: Display full node name. Default: Disable
        • Enable Message Bubbles
          Bool: Show messages as bubbles. Default: Enable
      • LoRa
        • Use Preset
          Bool: Use predefined modem presets instead of custom. Default: Enable
        • Modem Preset
          Enum: LONG_FAST, SHORT_TURBO, etc., for bandwidth/speed trade-off. Default: LONG_FAST
        • Region
          Enum: UNSET, US, EU_868, etc., for regulatory compliance. Default: UNSET
        • Bandwidth
          uint32: LoRa bandwidth in Hz. Default: Preset-dependent
        • Spread Factor
          uint32: Spread factor (7-12) for range vs. speed. Default: Preset-dependent
        • Coding Rate
          uint32: Coding rate denominator (5-8) for error correction. Default: Preset-dependent
        • Frequency Offset
          Float: Calibration offset for frequency. Default: 0
        • Hop Limit
          uint32: Max hops for messages (default: 3, max 7). Default: 3
        • TX Enabled
          Bool: Enable radio transmission. Default: Enable
        • TX Power
          int32: Transmit power in dBm (0 for max legal). Default: Max legal
        • Channel Num
          uint32: Channel number for frequency hopping. Default: 0
        • Override Duty Cycle
          Bool: Ignore duty cycle limits (use with caution). Default: Disable
        • SX126X RX Boosted Gain
          Bool: Boost RX gain on SX126X chips. Default: Disable
        • Ignore Incoming
          Repeated uint32: Ignore packets from these node IDs. Default: Empty
        • Override Frequency
          Float: Override center frequency. Default: 0
        • Ignore MQTT
          Bool: Ignore MQTT-sourced packets. Default: Disable
        • PA Fan Disabled
          Bool: Disable PA fan for cooling. Default: Disable
      • Bluetooth
        • Enabled
          Bool: Enable Bluetooth for app connections. Default: Enable
        • Mode
          Enum: RANDOM_PIN, FIXED_PIN, NO_PIN for pairing security. Default: RANDOM_PIN
        • Fixed Pin
          uint32: Fixed pairing PIN if mode is FIXED_PIN. Default: 123456
      • Security
        • Serial Enabled
          Bool: Enable serial console for debug (disables StreamAPI if off). Default: Enable
        • Debug Log Enabled
          Bool: Keep debug logs active even with API client connected. Default: Disable
        • Public Key
          Bytes: Public key for encryption. Default: Generated
        • Private Key
          Bytes: Private key for encryption. Default: Generated
      • Module Settings (Each module has its own submenu)
        • MQTT
          • Enabled
            Bool: Enable MQTT client for internet gateway. Default: Disable
          • Address
            String: MQTT server address (e.g., mqtt.meshtastic.org). Default: mqtt.meshtastic.org
          • Username
            String: MQTT username for authentication. Default: Empty
          • Password
            String: MQTT password. Default: Empty
          • Encryption Enabled
            Bool: Encrypt MQTT traffic. Default: Enable
          • JSON Enabled
            Bool: Use JSON format for MQTT messages. Default: Disable
          • TLS Enabled
            Bool: Use TLS for secure MQTT. Default: Enable
          • Root
            String: MQTT root topic (default: 'msh'). Default: msh
          • Proxy To Client Enabled
            Bool: Proxy mesh to MQTT client. Default: Disable
          • Map Reporting Enabled
            Bool: Enable map position reporting via MQTT. Default: Disable
          • Map Report Settings
            Sub: Publish Interval Secs, Position Precision, Should Report Location. Default: 3600 secs, 32 bits, Enable
        • Serial
          • Enabled
            Bool: Enable serial module for external interfaces. Default: Disable
          • Echo
            Bool: Echo serial input for testing. Default: Disable
          • RXD
            uint32: RX pin for serial. Default: Hardware-specific
          • TXD
            uint32: TX pin for serial. Default: Hardware-specific
          • Baud
            Enum: 9600, 115200, etc., for baud rate. Default: 38400
          • Timeout
            uint32: Serial timeout in seconds. Default: 1
          • Mode
            Enum: DEFAULT, SIMPLE, PROTO, TEXTMSG, NMEA, etc., for serial mode. Default: DEFAULT
          • Override Console Serial Port
            Bool: Override default console serial port. Default: Disable
        • External Notification
          • Enabled
            Bool: Enable external notifications (LED, buzzer, etc.). Default: Disable
          • Output Ms
            uint32: Notification duration in ms. Default: 1000
          • Output
            uint32: GPIO for output. Default: Hardware-specific
          • Output Vibra
            uint32: GPIO for vibration motor. Default: Hardware-specific
          • Output Buzzer
            uint32: GPIO for buzzer. Default: Hardware-specific
          • Active
            Bool: Active high/low for notifications. Default: High
          • Alert Message
            Bool: Alert on messages. Default: Enable
          • Alert Message Vibra
            Bool: Vibrate on messages. Default: Enable
          • Alert Message Buzzer
            Bool: Buzz on messages. Default: Enable
          • Alert Bell
            Bool: Alert with bell sound. Default: Enable
          • Alert Bell Vibra
            Bool: Vibrate on bell. Default: Enable
          • Alert Bell Buzzer
            Bool: Buzz on bell. Default: Enable
          • Use PWM
            Bool: Use PWM for buzzer control. Default: Enable
          • Nag Timeout
            uint32: Timeout for nagging alerts. Default: 60
          • Use I2S As Buzzer
            Bool: Use I2S interface as buzzer. Default: Disable
        • Store & Forward
          • Enabled
            Bool: Enable store & forward for offline messaging. Default: Disable
          • Heartbeat
            Bool: Send heartbeat for presence. Default: Enable
          • Records
            uint32: Number of message records to store. Default: 100
          • History Return Max
            uint32: Max history messages to return. Default: 50
          • History Return Window
            uint32: Time window for history return. Default: 3600
          • Is Server
            Bool: Act as server for store & forward. Default: Disable
        • Range Test
          • Enabled
            Bool: Enable range test module. Default: Disable
          • Sender
            uint32: Interval for sending test packets. Default: 30
          • Save
            Bool: Save test results to file. Default: Enable
          • Clear On Reboot
            Bool: Clear results on reboot. Default: Enable
        • Telemetry
          • Device Update Interval
            uint32: Device metrics update interval. Default: 300
          • Environment Update Interval
            uint32: Environment sensor update interval. Default: 300
          • Environment Measurement Enabled
            Bool: Enable environment measurements (temp, humidity). Default: Disable
          • Environment Screen Enabled
            Bool: Show environment on device screen. Default: Disable
          • Environment Display Fahrenheit
            Bool: Display temp in Fahrenheit. Default: Disable
          • Air Quality Enabled
            Bool: Enable air quality sensor. Default: Disable
          • Air Quality Interval
            uint32: Air quality update interval. Default: 300
          • Power Measurement Enabled
            Bool: Enable power monitoring. Default: Disable
          • Power Update Interval
            uint32: Power metrics update interval. Default: 300
          • Power Screen Enabled
            Bool: Show power metrics on screen. Default: Disable
          • Health Measurement Enabled
            Bool: Enable health monitoring. Default: Disable
          • Health Update Interval
            uint32: Health update interval. Default: 300
          • Health Screen Enabled
            Bool: Show health on screen. Default: Disable
          • Device Telemetry Enabled
            Bool: Enable general device telemetry. Default: Disable
          • Air Quality Screen Enabled
            Bool: Show air quality on screen. Default: Disable
        • Canned Message
          • Rotary1 Enabled
            Bool: Enable rotary encoder for canned messages. Default: Disable
          • Inputbroker Pin A
            uint32: Pin A for input broker. Default: Hardware-specific
          • Inputbroker Pin B
            uint32: Pin B for input broker. Default: Hardware-specific
          • Inputbroker Pin Press
            uint32: Pin for press event. Default: Hardware-specific
          • Inputbroker Event CW
            Enum: UP, DOWN, etc., for clockwise event. Default: UP
          • Inputbroker Event CCW
            Enum: UP, DOWN, etc., for counterclockwise. Default: DOWN
          • Inputbroker Event Press
            Enum: SELECT, etc., for press event. Default: SELECT
          • Updown1 Enabled
            Bool: Enable up/down buttons for messages. Default: Disable
          • Send Bell
            Bool: Send bell with messages. Default: Disable
        • Audio (Voice/Codec2)
          • Codec2 Enabled
            Bool: Enable Codec2 voice encoding. Default: Disable
          • PTT Pin
            uint32: Push-to-talk pin. Default: Hardware-specific
          • Bitrate
            Enum: CODEC2_3200, etc., for bitrate. Default: CODEC2_3200
          • I2S WS
            uint32: I2S word select pin. Default: Hardware-specific
          • I2S SD
            uint32: I2S serial data pin. Default: Hardware-specific
          • I2S DIN
            uint32: I2S data in pin. Default: Hardware-specific
          • I2S SCK
            uint32: I2S clock pin. Default: Hardware-specific
        • Remote Hardware
          • Enabled
            Bool: Enable remote GPIO control. Default: Disable
          • Allowed
            Bool: Allow remote hardware access. Default: Disable
          • Available Pins
            Repeated: List of available GPIO pins. Default: Empty
          • GPIO Watch
            uint32: Pin to watch for changes. Default: 0
        • Neighbor Info
          • Enabled
            Bool: Enable neighbor information sharing. Default: Disable
          • Update Interval Secs
            uint32: Update interval for neighbor data. Default: 300
        • Ambient Lighting
          • Current
            uint32: LED current (default: 10). Default: 10
          • Red
            uint32: Red level (0-255). Default: 0
          • Green
            uint32: Green level (0-255). Default: 0
          • Blue
            uint32: Blue level (0-255). Default: 0
        • Detection Sensor
          • Enabled
            Bool: Enable motion/detection sensor. Default: Disable
          • Minimum Broadcast Secs
            uint32: Min broadcast interval for detections. Default: 30
          • State Broadcast Secs
            uint32: State change broadcast interval. Default: 300
          • Send Bell
            Bool: Send bell on detection. Default: Disable
          • Name
            String: Sensor name. Default: Empty
          • Monitor Pin
            uint32: Pin to monitor for triggers. Default: Hardware-specific
          • Detection Triggered High
            Bool: Trigger on high signal. Default: Enable
          • Use Input 2
            Bool: Use secondary input. Default: Disable
          • Alert Message Enabled
            Bool: Enable alert messages. Default: Enable
          • Alert Message Vibra
            Bool: Vibrate on alerts. Default: Enable
          • Alert Message Buzzer
            Bool: Buzz on alerts. Default: Enable
          • Alert Bell Enabled
            Bool: Enable bell alerts. Default: Enable
          • Use PWM
            Bool: Use PWM for alerts. Default: Enable
        • Paxcounter (People counter)
          • Enabled
            Bool: Enable people counting via WiFi/BLE scans. Default: Disable
          • WiFi Scan Interval
            uint32: WiFi scan interval. Default: 120
          • BLE Scan Interval
            uint32: BLE scan interval. Default: 120
          • Send Update Interval
            uint32: Send count update interval. Default: 300
          • Screen Enabled
            Bool: Show counts on screen. Default: Disable
        • RTTTL Buzzer (Ring tones)
          • Enabled
            Bool: Enable RTTTL melody playback. Default: Disable
          • RTTTL
            String: RTTTL melody string (e.g., 'd=4,o=5,b=140:8e6'). Default: Empty
        • TAK (For ATAK integration)
          • Enabled
            Bool: Enable TAK/ATAK integration module. Default: Disable
          • PLI Interval Secs
            uint32: Position Location Information (PLI) broadcast interval. Default: 60
        • Traffic Management (Experimental)
          • Enabled
            Bool: Enable experimental traffic management features. Default: Disable
        • Status Message (Periodic status)
          • Enabled
            Bool: Enable periodic status messages. Default: Disable
          • Interval Secs
            uint32: Interval for status broadcasts. Default: 3600
  • Debug
    • Logs
      View system logs for troubleshooting. Default: N/A
    • Factory Reset
      Reset device to factory settings. Default: N/A
    • Share Debug Info
      Share debug information for support. Default: N/A
